Co to je dialogový systém?

May 2, 2018

Dialogový systém je počítačový program, jehož účelem je komunikovat s člověkem pomocí mluvené řeči. Hlasové dialogové systémy mají široké možnosti uplatnění, kterými jsou např. automatizace hlasových kontaktních center, interaktivní zábava, výuka jazyků, pomoc hendikepovaným a aplikace, kde jsou jiné ovládací a prezentační možnosti zařízení omezené Typicky se tyto systémy skládají z těchto komponentů: rozpoznání mluvené řeči porozumění přirozenému jazyku řízení dialogu generování odezvy syntéza mluvené řeči
